# Contact Manager Spring Boot

Fullstack Contact Manager with Spring Boot, PostgreSQL, Docker and a simple JavaScript frontend.

## Features

- Create contacts
- Show all contacts
- Update contacts
- Delete contacts
- Toggle favorite status
- Validation with custom error responses
- PostgreSQL persistence
- REST API tests

## Tech Stack

- Java 17
- Spring Boot
- Spring Web
- Spring Data JPA
- PostgreSQL
- Docker Compose
- Maven
- JavaScript
- HTML/CSS
- JUnit / MockMvc

## Run locally

```bash
docker compose up -d
mvn spring-boot:run
